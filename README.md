# Age Calculator

This is my first Python project — a simple age calculator.

It asks the user for their name and birth year, calculates their age, and prints fun messages based on their age.

### How to run
1. Clone the repo.
2. Run `python age_calculator.py` in a Python environment.
3. Enter your details when prompted.

### Skills used
- Python basics (input, print, conditions)
- Loops
- User interaction
